# JAVA ile Makine Öğrenmesi Algoritmaları

Bu repo, **BTK Akademi için hazırlanan JAVA ile Makine Öğrenmesi Algoritmaları** eğitiminde hazırlanan uygulama kodlarını ve bölüm bazlı örnekleri içerir.


## Eğitim Hakkında

Bu eğitimde makine öğrenmesinin temel kavramlarından başlayarak gözetimli/gözetimsiz öğrenme, regresyon, sınıflandırma, topluluk öğrenmesi, boyut indirgeme, pekiştirmeli öğrenme ve uçtan uca proje geliştirme adımları Java tabanlı kütüphaneler (özellikle WEKA ve SMILE) ile işlenmektedir.
